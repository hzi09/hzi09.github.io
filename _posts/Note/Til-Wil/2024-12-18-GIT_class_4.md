# .gitignore
## .gitignore란?
- .gitignore 파일이란 Git 버전 관리에서 제외할 파일 목록을 의미
- git으로 프로젝트를 관리할 때, 그 프로젝트 안의 특정파일을 Git으로 관리할 필요가 없는 경우
  - 예를 들면, 로그파일, 민감정보, 불필요한 시스템 파일 등등
- .gitignore은 git으로 관리한다면 꼭 있어야 함!!
