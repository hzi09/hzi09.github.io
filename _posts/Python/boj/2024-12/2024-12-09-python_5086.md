[🔗5086. 배수와 약수](https://www.acmicpc.net/problem/5086)

# 📝문제 요약
## 문제
4 × 3 = 12이다.

이 식을 통해 다음과 같은 사실을 알 수 있다.

3은 12의 약수이고, 12는 3의 배수이다.

4도 12의 약수이고, 12는 4의 배수이다.

두 수가 주어졌을 때, 다음 3가지 중 어떤 관계인지 구하는 프로그램을 작성하시오.

1. 첫 번째 숫자가 두 번째 숫자의 약수이다.
2. 첫 번째 숫자가 두 번째 숫자의 배수이다.
3. 첫 번째 숫자가 두 번째 숫자의 약수와 배수 모두 아니다.


## 입력
입력은 여러 테스트 케이스로 이루어져 있다. 각 테스트 케이스는 10,000이 넘지않는 두 자연수로 이루어져 있다. 마지막 줄에는 0이 2개 주어진다. 두 수가 같은 경우는 없다.

## 출력
각 테스트 케이스마다 첫 번째 숫자가 두 번째 숫자의 약수라면 factor를, 배수라면 multiple을, 둘 다 아니라면 neither를 출력한다.

<br>

# ✏️문제 풀이
- 문제풀이는 주석으로 처리

```python
while True :
    # 숫자 n1, n2 받기
    n1, n2 = map(int, input().split())
    # n1, n2가 0이면 실행 종료
    if n1 == 0 and n2 == 0 :
        break
    # n1이 n2보다 작으면
    elif n1 < n2 :
        # n2/n1의 나머지가 0일 경우는 factor 출력
        if n2 % n1 == 0 :
            print('factor')
    # n1이 n2보다 크거나 같으면
    elif n1 >= n2 :
        # n1/n2의 나머지가 0일 경우는 multiple 출력
        if n1 % n2 == 0:
            print('multiple')
        # n1/n2의 나머지가 0이 아닌 경우에는 neither 출력
        elif n1 % n2 != 0 :
            print('neither')
```

# 💯제출 코드
```python
while True :
    n1, n2 = map(int, input().split())
    if n1 == 0 and n2 == 0 :
        break
    elif n1 < n2 :
        if n2 % n1 == 0 :
            print('factor')
    elif n1 >= n2 :
        if n1 % n2 == 0:
            print('multiple')
        elif n1 % n2 != 0 :
            print('neither')
```
