# 📝문제 요약
## 문제
카지노에서 제일 인기 있는 게임 블랙잭의 규칙은 상당히 쉽다. 카드의 합이 21을 넘지 않는 한도 내에서, 카드의 합을 최대한 크게 만드는 게임이다. 블랙잭은 카지노마다 다양한 규정이 있다.

한국 최고의 블랙잭 고수 김정인은 새로운 블랙잭 규칙을 만들어 상근, 창영이와 게임하려고 한다.

김정인 버전의 블랙잭에서 각 카드에는 양의 정수가 쓰여 있다. 그 다음, 딜러는 N장의 카드를 모두 숫자가 보이도록 바닥에 놓는다. 그런 후에 딜러는 숫자 M을 크게 외친다.

이제 플레이어는 제한된 시간 안에 N장의 카드 중에서 3장의 카드를 골라야 한다. 블랙잭 변형 게임이기 때문에, 플레이어가 고른 카드의 합은 M을 넘지 않으면서 M과 최대한 가깝게 만들어야 한다.

N장의 카드에 써져 있는 숫자가 주어졌을 때, M을 넘지 않으면서 M에 최대한 가까운 카드 3장의 합을 구해 출력하시오.

## 입력
첫째 줄에 카드의 개수 N(3 ≤ N ≤ 100)과 M(10 ≤ M ≤ 300,000)이 주어진다. 둘째 줄에는 카드에 쓰여 있는 수가 주어지며, 이 값은 100,000을 넘지 않는 양의 정수이다.

합이 M을 넘지 않는 카드 3장을 찾을 수 있는 경우만 입력으로 주어진다.

## 출력
첫째 줄에 M을 넘지 않으면서 M에 최대한 가까운 카드 3장의 합을 출력한다.

<br>

# ✏️문제 풀이
- 문제풀이는 주석으로 처리

```python
# 랜덤으로 리스트에 있는 값들의 조합을 구하는 모듈 사용
from itertools import permutations

# 카드의 갯수 n과 카드의 합 m 
n, m = map(int, input().split())
# 각 카드의 값은 입력 받아 리스트로 저장
card_list = list(map(int, input().split()))
# 카드의 모든 조합을 저장할 빈 리스트 생성
card_group = []

# 3개씩 있는 모든 조합을 card_group 리스트에 저장
for i in list(permutations(card_list, 3)) :
    card_group.append(i)

# card_group리스트의 각 요소의 합계를 저장할 리스트
sum_card = []

# card_group의 요소만큼 반복
for i in range(len(card_group)) :
    # sum_num은 카드의 각 요소의 합
    sum_num = sum(card_group[i])
    # sum_num의 값이 m보다 작거나 같을 경우에만 sum_card 리스트에 요소를 sum_num 값을 저장
    if m >= sum_num : 
        sum_card.append(sum_num)

# sum_card 리스트의 가장 큰 값을 출력
print(max(sum_card))
```

from itertools import permutations

n, m = map(int, input().split())
card_list = list(map(int, input().split()))
card_group = []

for i in list(permutations(card_list, 3)) :
    card_group.append(i)

sum_card = []
for i in range(len(card_group)) :
    sum_num = sum(card_group[i])
    if m >= sum_num : 
        sum_card.append(sum_num)

print(max(sum_card))
