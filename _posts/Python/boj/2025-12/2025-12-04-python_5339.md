# 📝문제 요약
## 문제

스타워즈에 등장하는 로봇인 C3PO는 요즘 콜센터에 근무하고 있다. 콜센터에 앉아있는 C3PO를 그리는 프로그램을 작성하시오.

## 출력

예제 출력처럼 콜센터에 앉아있는 C3PO를 출력한다. 마지막 세 줄의 두 '|' 사이에는 공백이 10개 있다.


<br>

# ✏️문제 풀이
- docstring을 사용하여 해당 그림을 복사하여 붙혀 넣음
- 각줄에서 마지막에 있는 \는 붙혀쓰라는 의미가 되기 때문에 두개를 사용 → \\ : 1, 4, 5 번째 줄 수정

## 코드와 함께 보는 풀이

```python
# 그림을 붙혀 넣고 1, 4, 5번 째줄에 \만 추가하여 수정
c3po = """     /~\\
    ( oo|
    _\=/_
   /  _  \\
  //|/.\|\\\\
 ||  \ /  ||
============
|          |
|          |
|          |

"""

# 그림 출력
print(c3po)
```
