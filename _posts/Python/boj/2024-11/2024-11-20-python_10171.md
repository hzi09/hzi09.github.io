# 📝문제 요약
## 문제 설명
아래 예제와 같이 고양이를 출력하시오.

## 입력
없음.

## 출력
고양이를 출력한다.


<br>

# ✏️문제 풀이
- 처음에 print()함수 안에 줄바꿈을 위한 `\n` 를 사용하여 출력
  - 첫 줄의 \를 인식하지 못하여 한줄로 됨.
    
    ```python
    print("\    /\\n)  ( ')\n(  /  )\n\(__)|") 
    ```
    ```
    \    /\n)  ( ')  <- 여기서 오류가 났음
    (  /  )
    \(__)|
    ```
- `\`를 사용하기 위해 서는 `\\`를 사용하여야 함!

<br>

# 💯제출 코드
```python
print("\    /\\\n )  ( ')\n(  /  )\n \(__)|")
```
- 결과
    ```
    \    /\
    )  ( ')
    (  /  )
    \(__)|
    ```
