# 📝문제 요약
## 문제 설명
첫째 줄에는 별 1개, 둘째 줄에는 별 2개, N번째 줄에는 별 N개를 찍는 문제

하지만, 오른쪽을 기준으로 정렬한 별(예제 참고)을 출력하시오.

## 입력
첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.

## 출력
첫째 줄부터 N번째 줄까지 차례대로 별을 출력한다.


<br>

# ✏️문제 풀이
- 별 찍을 개수 변수 a 선언  `a = int(input())`
- for문을 사용하여 a만큼 반복 `for i in range(a):`
    - print() 함수를 사용하여 별 출력
        - ‘ ‘(띄어쓰기)를 a - (i + 1) 만큼 써주고
        - *는 i+1만큼  써준다.
        - i는 0부터라서 +1을 하는 것!

<br>

# 💯제출 코드
```python
a = int(input())

for i in range(a):
    print(' ' * (a-i-1), end ="")
    print("*" * (i+1))
```
