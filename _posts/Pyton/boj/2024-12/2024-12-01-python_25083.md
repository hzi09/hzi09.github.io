# 📝문제 요약
## 문제 설명
아래 예제와 같이 새싹을 출력하시오.

## 입력
입력은 없다.

## 출력
새싹을 출력한다.
```
         ,r'"7
r`-_   ,'  ,/
 \. ". L_r'
   `~\/
      |
      |
```


<br>

# ✏️문제 풀이
- ' 또는 "를 출력하기 위해서는 \표시를 눌러 해당 기호들 앞에 ＼넣기

<br>

# 💯제출 코드
```python
print("         ,r\'\"7")
print("r`-_   ,\'  ,/")
print(" \\. \". L_r\'")
print("   `~\\/")
print("      |")
print("      |")
```
